São

Por exemplo, se você precisa de um código que peça a idade do usuário e faça um cálculo, você pode escrever isso em forma de função, para não precisar escrever o cálculo do zero todas as vezes que for pedir a idade do usuário. Veja abaixo o exemplo em Python. 

                        def idadeMultiplicada():
                            idade = input("Digite sua idade: ")
                            return idade * 10
                    
   
                        idadeMultiplicada()

É criada a função idadeMultiplicada em que pede a idade do usuário e retorna ela multiplicada por 10. Se fosse necessário pedir a idade do usuário diversas vezes e fazer esse cálculo com ela, usando funções, não seria preciso fazer todas as linhas de código novamente, apenas chamar a função idadeMultiplicada novamente.

Parâmetros

Funções podem ter parâmetros, para poderem ficar mais dinâmicas. Parâmetros são como "entradas" para as funções. Veja abaixo

                        def soma(x, y):
                            return x + y

Na função soma(x, y) ela recebe dois parâmetros: x e y e retorna a soma deles. Dessa forma, é possível realizar a soma de números diferentes em diferentes chamadas da função

Para poder chamar uma função e passar parâmetros basta entre parênteses digitar os valores respectivos. No caso da função soma(x, y), ao chamar ela, é necessário informar qual será o valor de x e y dentro dos parênteses, como mostrado acima
