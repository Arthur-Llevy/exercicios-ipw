Listas

Lista é um tipo de dado que armazena literalmente uma lista, por exemplo, uma lista de números, entre outros. Elas são úteis quando você precisa armazenar muitas informações, lista de nomes por exemplo. Veja abaixo o exemplo em Python

nomes = ["Arthur", "Maria", "José"]

De forma simples, criamos uma lista de nomes.

Acessando seus valores

Para poder acessar um valor da lista, basta indicar em qual indice o item que você quer está. Lembrando que em maioria das linguagens de programação, o primeiro índice é zero. Veja a representação abaixo:

Veja como acessar o primeiro nome da lista nomes criada anteriormente

nomes[0]

O código acima irá exibir o nome "Arthur", pois este texto ocupa a posição 0
